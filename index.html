Animask

Animask framework is an open source software for Anomaly detection using Artificial Intelligence and Deep Learning. It offers a comprehensive tool for identifying outliers and unusual patterns in complex datasets. The framework incorporates state-of-the-art deep learning architectures, including autoencoders and generative adversarial networks, to model normal behavior and detect deviations.
